# Change Log

## 1.1.0 - 2026-06-30

### Added

- Added RegEx pattern matching to group references.
- Added RegEx pattern matching to template frame groups and variants.
- Added wild card matching to template frame groups and variants.
- Added input group color mapping.
- Added template frame transforms: trim, flip\_h, flip\_v, rotate\_right, and rotate\_left.

### Changed

- Group defined variants no longer needs to exist in an input image.
- Output config direction has been replaced with frame\_direction and sheet\_direction.
- Output config rows has been replaced with frame\_rows and sheet\_rows.
- Output config cols has been replaced with frame\_cols and sheet\_cols.
- Output config width has been replaced with frame\_width and sheet\_width.
- Output config height has been replaced with frame\_height and sheet\_height.

## 1.0.1 - 2025-12-13

### Added

- Added [[version]] output variable.

### Changed

- Specifying "Default" as the theme will use original colors.

## 1.0.0 - 2025-11-10

Initial release.
